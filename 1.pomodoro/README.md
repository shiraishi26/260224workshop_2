# ポモドーロタイマー

カスタマイズ可能なポモドーロタイマーWebアプリケーション

## 機能

### ⏱️ ポモドーロ時間のカスタマイズ
- 15分、25分、35分、45分から選択可能
- デフォルト: 25分

### ☕ 休憩時間のカスタマイズ
- 5分、10分、15分から選択可能
- デフォルト: 5分

### 🎨 テーマ切り替え
- **ライトモード**: 明るく快適な作業環境
- **ダークモード**: 目に優しい暗めの配色
- **フォーカスモード**: 集中力を高める紫系の配色

### 🔊 サウンド設定
- **開始音**: タイマー開始時の音
- **終了音**: ポモドーロ/休憩終了時の音
- **tick音**: 毎秒のtick音（オプション）

### その他の機能
- 完了したポモドーロ数の追跡
- プログレスリング表示
- ブラウザ通知（サポートされている場合）
- 設定の永続化（LocalStorage使用）
- キーボードショートカット（スペースキーで開始/一時停止）
- レスポンシブデザイン

## 使い方

### 起動方法

#### 方法1: Pythonのシンプルなサーバーを使用

```bash
cd 1.pomodoro
python3 app.py
```

ブラウザで `http://localhost:8000/index.html` にアクセス

#### 方法2: 標準のHTTPサーバーを使用

```bash
cd 1.pomodoro
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000/index.html` にアクセス

### 基本的な使い方

1. **タイマーを開始**: 「開始」ボタンをクリック、またはスペースキーを押す
2. **一時停止**: 「一時停止」ボタンをクリック、またはスペースキーを押す
3. **リセット**: 「リセット」ボタンをクリック
4. **設定を変更**: 右上の⚙️アイコンをクリック

### 設定の変更

1. 右上の⚙️アイコンをクリックして設定画面を開く
2. 希望の設定を選択
3. 「保存」ボタンをクリック

## 技術仕様

- **HTML5**: セマンティックなマークアップ
- **CSS3**: カスタムプロパティを使用したテーマシステム
- **JavaScript (ES6+)**: クラスベースのアーキテクチャ
- **Web Audio API**: サウンド生成
- **Notification API**: ブラウザ通知
- **LocalStorage API**: 設定の永続化
