# Pomodoro Timer - パターンA: 視覚的フィードバックの強化

## 概要

このプロジェクトは、Pomodoroタイマーに視覚的フィードバック機能を追加し、ユーザーの集中力を高めることを目的としています。

## 実装した機能

### 1. 円形プログレスバーのアニメーション
- SVGベースの円形プログレスバー
- 残り時間に応じて滑らかに減少
- レスポンシブデザイン対応

### 2. 時間経過による色変化
- **青フェーズ (100% - 50%)**: 集中開始時、落ち着いた青色
- **黄色フェーズ (50% - 20%)**: 中間段階、注意喚起の黄色
- **赤フェーズ (20% - 0%)**: 終了間近、警告の赤色

### 3. パーティクルエフェクト
- 集中時間中に白い粒子が浮遊
- ランダムなサイズと速度
- 上昇アニメーション

### 4. 波紋アニメーション
- ランダムな位置に波紋が出現
- 拡散していく視覚効果
- 没入感を高める演出

## セットアップ方法

1. 依存関係のインストール:
```bash
cd 1.pomodoro
pip install -r requirements.txt
```

2. アプリケーションの起動:
```bash
python app.py
```

3. ブラウザで以下のURLにアクセス:
```
http://localhost:5000
```

## ファイル構成

```
1.pomodoro/
├── app.py                    # Flask アプリケーション

└── static/

    │   └── style.css        # スタイルシート
    └── js/
        └── script.js        # JavaScript ロジック
```

## 使い方

1. **開始**: 「開始」ボタンをクリックして25分の集中時間を開始
2. **一時停止**: 必要に応じて「一時停止」ボタンで一時停止
3. **リセット**: 「リセット」ボタンでタイマーをリセット
4. **休憩**: 25分後に自動的に5分の休憩時間に切り替わり

## 技術スタック

- **バックエンド**: Flask 3.0.0
- **フロントエンド**: HTML5, CSS3, Vanilla JavaScript
- **アニメーション**: CSS Animations, SVG

## セキュリティ

- CodeQL スキャン完了: 脆弱性なし
- Flask debug モードを無効化

## テスト目的

視覚的な没入感がユーザーの集中力に与える影響を測定する。
